# Centerflex

Especialistas em conexões hidráulicas e industriais com atendimento rápido e entrega nacional.

## Sobre

A **Centerflex** é uma loja especializada em componentes para sistemas hidráulicos e industriais, oferecendo soluções completas para quem exige o melhor em conexões de alta pressão.

## Produtos

- **Mangueiras hidráulicas** — Mangueiras para sistemas de pressão, condução de óleo e aplicações hidráulicas exigentes.
- **Mangueiras industriais** — Opções para condução de ar, água e fluidos em rotinas industriais e de manutenção.
- **Abraçadeira** — Fixação segura para montagem, vedação e suporte em aplicações hidráulicas e industriais.
- **Anéis O-Ring** — Elementos de vedação para reduzir vazamentos e manter o desempenho do sistema.
- **Conexões de latão** — Peças resistentes para linhas de ar comprimido, com boa durabilidade e encaixe preciso.
- **Conexões porca e anel (DIN 2353)** — Solução DIN 2353 para montagens firmes, seguras e com excelente acabamento técnico.
- **Conexão de ar comprimido** — Conexões resistentes para sistemas pneumáticos e linhas de ar comprimido industrial.
- **Engates rápidos hidráulicos** — Conexão prática para acoplamento e desacoplamento de linhas hidráulicas com segurança e agilidade.
- **Graxeiras** — Componentes para lubrificação de máquinas, equipamentos e pontos de manutenção industrial.
- **Tubos de cobre** — Tubos resistentes para instalações, condução e aplicações técnicas com bom acabamento.
- **Conexões JIC** — Conexões JIC (Joint Industry Council) para sistemas hidráulicos de alta pressão com encaixe por 37º flare.
- **Válvulas esfera** — Válvulas para bloqueio e controle de fluxo em sistemas hidráulicos, pneumáticos e industriais.
